## 1. The 4-Phase Cycle
1. **Phase 1: Foundation (Drums)** - Full structure in EZDrummer.
2. **Phase 2: Core (Rhythm/Bass)** - Lock the groove with EZbass and Rhythm guitars.
3. **Phase 3: Voice (Lead)** - Record main melodies and solos.
4. **Phase 4: Polish (Ear Candy)** - Transitions, automation, and mixing.

---

## 2. Rituals

### Session Startup
1. Close all browser tabs except the reference song.
2. Open DAW and set BPM immediately.
3. Set a timer for 45 minutes and GO.

### Session Shutdown
1. Export current bounce to "Demos" folder.
2. Listen once away from the computer.
3. Write down the **one thing** to fix next session.
